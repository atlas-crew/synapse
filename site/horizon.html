Architecture
Why Edge Intelligence?
Traditional cloud WAFs add round-trip latency, become a single point of failure, and can't inspect response bodies without streaming data off-premises. Embedded intelligence keeps every decision local — at wire speed, with full context, and no third-party data exposure.
Traditional
Cloud WAF
  • 50–200ms round-trip — every request crosses the internet twice
  • Hard outage coupling — cloud provider down = your app down
  • Blind to response bodies — no DLP without streaming data out
  • Privacy exposure — payloads, tokens, and PII leave your perimeter
  • Cache poisoning surface — shared edge caches, noisy-neighbor risk
  • Detection lag — rule pushes take minutes to propagate globally
Synapse
Edge Intelligence
  • <1ms local decision — detection runs in-process with your proxy
  • Zero external dependencies — protection continues if Horizon is offline
  • Inline response inspection — DLP scans every byte before it leaves
  • Data sovereignty — payloads never leave the host, only anonymized signals
  • Isolated cache per tenant — no shared state, no cross-contamination
  • ~240μs hot reload — rule updates take effect with zero dropped requests
The tradeoff isn't latency vs accuracy — it's centralized control vs resilience. Horizon solves the first without sacrificing the second: the control plane orchestrates fleet-wide intelligence while each Synapse sensor remains independently authoritative at its own edge.

    Active Defense
    Interrogator System
    Progressive challenge escalation separates bots from humans without blocking legitimate traffic. Suspicious actors face increasing friction; clean traffic flows unimpeded.
    Challenge Progression
  • Level 1: Cookie Challenge — set tracking cookie, verify acceptance
  • Level 2: JavaScript PoW — SHA-256 proof-of-work computation
  • Level 3: CAPTCHA — human verification challenge
  • Level 4: Tarpit — progressive delays: 1.5^level (max 30s)
    • Honeypot Traps
    Configurable trap paths that instantly max risk for any actor that touches them.
    /.env, /wp-admin, /admin/config
    // Hit = instant risk 100
    De-escalation: Good behavior reduces challenge level. Legitimate users who pass challenges get remembered and flow freely on subsequent requests.

    Deploy
    Deployment & Integration
    Synapse runs wherever your traffic terminates. Choose the mode that matches your architecture — from a single-binary WAF in front of a monolith to a fleet of sensors reporting to a self-hosted Horizon control plane.
    Standalone WAF
    Single 25MB Binary
    Drop Synapse in front of any HTTP app. No control plane required — all detection, correlation, and blocking happens locally. Ideal for monoliths, internal tools, and air-gapped deployments.
    DOCKER · SYSTEMD · BARE METAL
    Fleet Mode
    Horizon + N Sensors
    Deploy Synapse sensors at every edge, reporting to a central Horizon control plane. Cross-sensor campaign correlation, centralized rule distribution, live threat map, and shared blocklists across the fleet.
    KUBERNETES · HELM · TERRAFORM
    Embedded Library
    @atlascrew/apparatus-lib
